Bulk Import Students from Excel

Instructions

  1. Prepare an Excel file (.xlsx or .xls) with the following columns:
    • Required columns: student_id, first_name, last_name, grade
    • Optional columns: division, room, section, student_type, email, phone, address
  2. Column names are case-insensitive (e.g., "Student_ID" or "student_id" both work)
  3. For student_type: use "hostel" or "day_scholar" (default: day_scholar)
  4. For division and room: use exact names that exist in the system
  5. Grade must be between 1 and 12
  6. Duplicate student IDs will be skipped
